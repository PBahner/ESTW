 Stellpult
===========
Für die andere hälfte der Modelleisenbahn gibt es ein Stellpult. Die "Kästchen" sind mit dem 3D-Drucker gedruckt und werden mit WS2811 ICs angesteuert.

![neues Gleisbildpult](Gleisbildpult/GBS_neu.jpg)
